# Furry Baby
A responsive, multi-page website built with HTML5 and CSS3 as part of my web development coursework at St. Petersburg College.

# About the Project
Furry Baby is a fictional pet-focused website designed to practice core front-end development skills including mobile-first responsive design, semantic HTML structure, and accessible UI layout.

# Built With
- HTML5
- CSS3
- Mobile-first responsive design

# Features
- Responsive layout across desktop, tablet, and mobile viewports
- Semantic HTML structure for accessibility and SEO
- WCAG accessibility standards applied throughout
- Clean navigation and visual hierarchy

# What I Learned
This project helped me practice structuring multi-page websites, writing clean semantic markup, and applying CSS layout techniques to create a consistent experience across screen sizes.
